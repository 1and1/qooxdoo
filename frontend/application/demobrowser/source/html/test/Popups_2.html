Test for the qx.legacy.ui.popup.Popup widget. qx.legacy.ui.popup.Popups are planned as the source constructor for tooltips, menus, ...

There is also a qx.legacy.ui.popup.PopupManager which hides all registered qx.legacy.ui.popup.Popup instances which are configured to auto hide. This is a nice method to omit to show more than one qx.legacy.ui.popup.Popup together (omit multiple QxTooltips, ...).

In this example we use a qx.legacy.ui.popup.PopupAtom. This is a popup with an automatically included qx.legacy.ui.basic.Atom, which allowes you to easily use it for messages to the user or something like this.
