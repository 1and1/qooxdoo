Some tests for some of the most used properties of qx.legacy.ui.core.Widget (coordinates, dimensions, backgroundColor, opacity, ...).

Also here are the first test of some widgets which extend qx.legacy.ui.core.Widget: qx.legacy.ui.form.TextField and qx.legacy.ui.form.PasswordField. These extended widgets also interacts like the basic QxWidgets with the qx.legacy.event.handler.FocusHandler.
