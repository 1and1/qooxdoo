Slider
******
The Slider widget is the classic widget for controlling a bounded value.

Preview Image
-------------
|Slider|

.. |Slider| image:: /pages/widget/slider.png

Features
--------
  * Mouse support.
  * Horizontal and vertical orientation.
  * Configurable steps.

Description
-----------
The Slider widget is the classic widget for controlling a bounded value. It lets the user move a slider handle along a horizontal or vertical groove and translates the handleâ€™s position into an integer value within the defined range.
